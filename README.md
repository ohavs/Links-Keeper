# 🔗 Links Keeper - שומר הלינקים שלי

אפליקציית PWA לניהול ושמירת לינקים, בנויה עם React + Firebase.

## ✨ פיצ'רים

- 📌 שמירת לינקים עם כותרת, תיאור, תגיות וקטגוריה
- 🏷️ פילטור לפי תגיות
- 📂 קטגוריות מותאמות אישית
- 🔍 חיפוש מהיר
- 🖱️ גרור ושחרר בין קטגוריות
- 📱 מותאם למובייל - PWA
- ☁️ סנכרון בענן עם Firebase Firestore

## 🛠️ טכנולוגיות

- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- Firebase Firestore
- Firebase Hosting
- Lucide Icons
- DnD Kit

## 🚀 התחלה מהירה

```bash
npm install
npm run dev
```

## 📦 דיפלוי

```bash
npm run build
firebase deploy
```
